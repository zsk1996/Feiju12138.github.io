命令行艺术文本框

## 安装

``` sh
brew install boxes
```

## 基本命令

### 默认方式

> `<text>`：需要输出的文字

``` sh
echo "<text>" | boxes
```

### 举牌狗

``` sh
echo "<text>" | boxes -d dog
```

### 举牌猫

``` sh
echo "<text>" | boxes -d cat
```
