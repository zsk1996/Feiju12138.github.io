## 日志文件位置

- `/var/log/crond`

## 基本命令

### 编辑

``` sh
crontab -e
```

#### 编写任务

- 连续的多个日期，用`-`连接
- 不连续的多个日期，用`,`隔开
- 当前日期的全部值，用`*`表示
- 日期频率（每`<minute>`分钟执行一次），用`*/<num>`表示

> `<minute>`：分钟数。0\~59的整数
> `<hour>`：小时（24小时制）。0\~23的整数
> `<day>`：天。1\~31的整数
> `<month>`：月。1\~12的整数
> `<week>`：星期。0\~7的整数（`0`和`7`都表示星期日）
> `<script>`：需要执行的脚本路径

``` sh
<minute> <hour> <day> <month> <week> <script>
```

### 查看

``` sh
crontab -l
```

### 清除

``` sh
crontab -r
```

### 指定用户

- 如果不指定用户，默认为当前用户

> `<user>`：用户名

``` sh
crontab -u <user>
```
