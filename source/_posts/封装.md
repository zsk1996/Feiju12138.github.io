## 封装

- 将类封装，就是将类中的成员变量不对外公开，使用`private`修饰，同时对外提供对成员变量操作的方法（get、set方法）

> `<class>`：类名

``` sh
public class <class> {
    
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
    
}
```
