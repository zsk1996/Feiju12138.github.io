## 前言

Linux挂载学习笔记

`mount`挂载操作，可以让目录成为设备的访问点

<!-- more -->

## 查看插入的设备

``` sh
ls -l /dev/cdrom
```

## 新建一个挂载点

``` sh
mkdir /dvd
```

## 设置挂载点

> `<dvd_src>`：设备路径
> `<dir_src>`：挂载点目录

``` sh
mount <dvd_src> <dir_src>
```

- mount: /dev/sr0 写保护，将以只读方式挂载

### 添加参数

> `<option>`：添加的参数

``` sh
mount -o <option_1>,<option_2> <dvd_src> <dir_src>
```

## 卸载

``` sh
umount <dir_src>
```

### 强制卸载

- 报错：`umount: /dvd：目标忙。`

``` sh
umount -f <dir_src>
umount -l <dir_src>
```
