## 前言

Swift Playgrounds 是一款适用于 Mac 和 iPad 的革命性 App，它使得代码学习和探索变得妙趣横生。“学习编程”提供了一系列课程，指导你通过互动学习来闯关，逐步掌握编程的基础知识。你还可以尝试众多挑战，从中探索许多独一无二的编程经验

## 学习编程1

### 执行函数

> `<name>`：函数名

``` swift
<name>()
```

### 创建函数

``` swift
func <name>() {
    ...
}
```

### for循环

- 从1开始循环

> `<num>`：循环的次数

``` swift
for i in 1 ... <num> {
    ...
}
```

### if判断

> `<boolean>`：一个函数返回的布尔值

``` swift
if <boolean> {
    ...
} else if <boolean> {
    ...
} else {
    ...
}
```

### 逻辑运算符

- 与`&&`
- 或`||`
- 非`!`

### while循环

``` swift
while <boolean> {
    ...
}
```
