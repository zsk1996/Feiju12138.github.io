## 前言

Java通过Timer实现任务调度

<!-- more -->

## 延时执行

- 到指定时间开始执行任务

``` java
Timer timer=new Timer();
timer.schedule(task, time);
```

### 示例

> `<time>`：定义开始执行的时间

``` java
Timer timer=new Timer();
timer.schedule(new TimerTask() {
    @Override
    public void run() {
        ...
        timer.cancel(); // 退出任务调度
    }
}, <time>);
```

## 周期执行

- 从指定时间开始，隔段时间，执行一次任务

``` java
Timer timer=new Timer();
timer.scheduleAtFixedRate(task, firstTime, period);
```
