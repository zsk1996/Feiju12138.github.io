## 要点一

- 计算结果的数据类型，与最大类型一致

## 要点二

- byte,short,char三种比int小的整数，运算时会先自动转换成int

``` java
byte a = 1;
byte b = 2;
byte c = (byte) (a + b);
```

## 要点三

- 超出范围的运算,会出现溢出现象
