## 前言

在用喇叭或音箱重复一段广告时，通常人们会选择直接录音，但是如果不是受过专业的训练，很难快速录制一个完美的录音，通过`MacOS`的`say`命令，可以让Siri阅读文字，生成语音文件

<!-- more -->

## Siri直接阅读文字

``` bash
say hello
```

## 生成语音文件

- 创建一个`.txt`文件，输入需要生成语音的内容保存

``` bash
touch Hello.txt
echo Hello >> Hello.txt
```

- 将`.txt`文件转换为`.m4a`文件

``` bash
say -f Hello.txt -o Hello.aiff
```

## 完成

最后将`.aiff`转换为`.mp3`就好啦
