## 源代码

``` sh
class Solution {
    public boolean isPalindrome(int x) {

        String str = x+"";
        StringBuilder sb = new StringBuilder(str);
        String newStr = sb.reverse().toString();
        if (newStr.equals(str)) {
            return true;
        }else {
            return false;
        }

    }
}
```
