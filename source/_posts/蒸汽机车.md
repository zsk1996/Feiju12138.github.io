## 前言

在命令行打印蒸汽机车（Steam Locomotive）

<!-- more -->

## 安装

``` sh
brew install sl
```

## 基本命令

> `-a`：似乎发生了意外。你会为那些哭喊求助的人们感到难过
> `-l`：显示小一点的火车
> `-F`：它居然飞走了
> `-e`：允许被`ctrl`+`c`中断（事实上`MacOS`并不可以）

``` sh
sl
```
