## 前言

用`Python`输出斐波那契数列

<!-- more -->

## 源代码

``` python
#!/usr/bin/env python

# 初始化斐波那契数列前两个数
list = [1, 1]

# 设置输出斐波那契数列长度
num = int(input("请输入长度: "))

# 生成斐波那契数列
for i in range(num-2):
    list.append(list[-2] + list[-1])

# 打印斐波那契数列
print(list)
```
