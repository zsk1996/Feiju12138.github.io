## 前言

命令行艺术字生成器

<!-- more -->

## 安装

``` sh
brew install toilet
```

## 基本命令

### 默认方式

> `<text>`：需要输出的文字

``` sh
toilet <text>
```

### 双色字

``` sh
toilet -f mono12 -F metal <text>
```

### 彩色字

``` sh
toilet -f mono12 -F gay <text>
```
