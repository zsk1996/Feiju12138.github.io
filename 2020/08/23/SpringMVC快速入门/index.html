【代码】SpringMVC快速入门

发表于 2020-08-23 分类于 Java

前言

SpringMVC快速入门

创建项目

  • 创建maven项目，配置pom.xml文件来引入项目所需要的jar包

配置前端控制器

  • 在web.xml文件内配置前端控制器

配置springmvc-config.xml

  • 在/src/main/resources/目录下创建springmvc-config.xml文件

实现Controller类

  • 在/src/main/java目录下创建普通的java类，通过注解的方式将其标记为Controller

创建测试jsp

  • 在/src/main/webapp/WEB-INF目录下新建/jsp/home.jsp文件

完成

  • 访问路径http://localhost:8080/项目名/Controller类的访问名/Controller内方法的访问名
