## 原理

- 修改`命令提示符`文字颜色

- 输出一段文字

- 无限循环

## 闪屏

- 创建一个`.bat`批处理文件

- 右键->编辑->写入代码->保存

``` bash
color 1a
echo 这是一段文字
color 2b
echo 这是一段文字
color 3c
echo 这是一段文字
color 4d
echo 这是一段文字
color 5f
echo 这是一段文字
color 6e
echo 这是一段文字
%0
```

- 右键->打开
