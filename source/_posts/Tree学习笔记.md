## 前言

`Tree`是一个能列出递归目录的命令，以图形显示驱动器或路径的文件夹结构

<!-- more -->

## 基本命令

- 显示当前目录下所有文件的树状结构

``` bash
tree
```

- 只显示部分层级的树状结构

``` bash
tree -L <num>
```

- 只显示目录

``` bash
tree -d
```

- 过滤不想显示的文件或目录

``` bash
tree -l <filename>
```
