## 前言

使用`lsof`查看端口号占用情况

<!-- more -->

## 查看所有端口占用情况

``` sh
lsof
```

### 分页显示

``` sh
lsof | less
```

## 查看指定端口占用情况

> `<port>`：端口号

``` sh
lsof -i :<port>
```
